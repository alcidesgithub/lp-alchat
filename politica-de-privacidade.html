Política de Privacidade

Última atualização: Janeiro de 2026

Esta Política de Privacidade descreve como o AlChat (operado por 52.753.253 ALCIDES CURSINO DE SIQUEIRA NETO) coleta, usa e protege as informações dos usuários que utilizam nossa plataforma de integração com as APIs da Meta (WhatsApp Business API, Instagram Graph API e Facebook Graph API).

2. Dados Coletados via APIs da Meta

Para fornecer o serviço de gestão de conversas omnichannel, o aplicativo AlChat solicita permissões para acessar os seguintes dados através das APIs oficiais da Meta:

  • Informações de Perfil Público: Nome, foto de perfil e ID de usuário do Facebook/Instagram/WhatsApp.
  • Conteúdo de Mensagens: Texto, imagens, vídeos e documentos enviados pelos clientes para as contas comerciais conectadas à plataforma.
  • Metadados da Interação: Horário de envio, status de leitura e número de telefone (no caso do WhatsApp).
Declaração de Uso de Dados da Plataforma Meta:
O AlChat utiliza os dados recebidos via APIs (WhatsApp, Instagram, Facebook) exclusivamente para permitir que a empresa contratante responda às mensagens de seus clientes através de nossa interface unificada. Não utilizamos esses dados para fins de vigilância, não vendemos dados a terceiros e não utilizamos o conteúdo das mensagens para fins publicitários.

3. Finalidade do Tratamento

Os dados são tratados estritamente para:

  • Centralizar o atendimento ao cliente em uma interface única.
  • Gerar métricas de produtividade (tempo de resposta, volume de atendimentos).
  • Manter histórico de conversas para consulta da própria empresa contratante.

4. Compartilhamento de Dados

Não compartilhamos seus dados pessoais com terceiros, exceto:

  • Meta Platforms, Inc.: O compartilhamento é inerente ao funcionamento das APIs oficiais para envio e recebimento de mensagens.
  • Obrigação Legal: Mediante ordem judicial ou requisição de autoridades competentes.

5. Retenção e Exclusão de Dados

Os dados das conversas são armazenados enquanto o contrato de prestação de serviços estiver ativo.

Direito de Exclusão: De acordo com a LGPD e as Políticas de Desenvolvedor da Meta, o usuário pode solicitar a exclusão completa de seus dados a qualquer momento. Para exercer este direito, envie uma solicitação para o e-mail ou WhatsApp abaixo. Os dados serão removidos de nossos servidores em até 72 horas após a confirmação.

6. Segurança

Utilizamos criptografia em trânsito (HTTPS/TLS) para toda a comunicação entre nossos servidores e as APIs da Meta, além de controles de acesso rigorosos em nosso banco de dados.
